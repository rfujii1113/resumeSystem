## 経歴書システム

### 概要
本システムは社内で経歴書の作成.編集.管理をシステム化し業務を簡素化します

### 主な機能
- ユーザーのログイン・登録機能
- 管理者によるユーザー管理
- 経歴書の作成編集
- 経歴書のPDF出力
- 経歴書検索

### 環境構築
1. **実行環境**
   - Java 17 以上
   - PostgreSQL
   - Springboot framework 3.x
   - Thymeleaf 3.x
